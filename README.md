## 📦 Getting Started

Follow these steps to run the project locally:

### 1. Clone the repo

```bash
git clone https://github.com/leonbadass/portfolio.git
cd portfolio
```
### 2. Install dependencies
```bash 
npm install
```

### 3. Start the development server
```bash
npm start
```

The app will be available at:
🔗 http://localhost:3000
